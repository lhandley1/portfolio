    ZeroWasteCook - AI Food Waste & Budgeting App

    Save More 💰 | Waste Less ♻️ | Cook Smart 🍳

    Problem:

    University students often struggle with food waste and tight budgets, contributing to financial strain and environmental impact. This project aimed to develop an AI-driven solution to help students minimise waste and save money through smarter recipe planning and sustainable cooking habits.


    Solution:

    Leveraging Agile (Scrum) principles, I developed ZeroWasteCook, an AI-powered web application for intelligent recipe recommendations and effective food management. The app classifies ingredients from user-uploaded images using a fine-tuned PyTorch InceptionV3 (CNN) model and sources filtered recipes via the Edamam's Recipe Search API.


    Key Features:

    • 🥦 Ingredient Recognition
    • 📅 Recipe Expiry Date Tracking
    • 🚫 Allergy & Preference Filters
    • 🔍 Smart Recipe Suggestions
    • 🍽️ Nutrition Insights
    • 🌱 Environmental Impact Rating - (A+ to F)
    • 💷 Estimated Food Waste Savings - (£)

    Core Technologies:

    • AI/Machine Learning: Python, PyTorch, InceptionV3 (CNN for Image Classification)
    • Web App Framework: Django
    • API Integration: Edamam Recipe Search API
    • Development Methodology: Agile, Scrum, BDD (Behaviour-Driven Development)
    • Dataset: Utilised the 'Food Classification' Kaggle dataset (61 food ingredient classes) for InceptionV3 model fine-tuning.

    Achievements & Impact:

    • Delivered a functional AI app built to reduce student food waste and inspire recipes, with its AI image classification model achieving 46.27% accuracy.
    • User research confirmed significant student interest (>70%) in key features, highlighting the app's potential for promoting sustainable and budget-friendly food habits.

    ChatAdvisor - AI Student Support Chatbot

    Problem:

    University students, especially those with learning difficulties, often face challenges in navigating extensive support resources and accessing timely, understandable advice. This project aimed to enhance information accessibility and provide a streamlined student support solution.


    Solution:

    In an Agile Scrum team of seven, we developed ChatAdvisor, an AI-powered chatbot leveraging Natural Language Processing (NLP) techniques. This conversational AI solution was designed for enhanced student support, providing quick access to university resources through summarised, easy-to-read information and personalised recommendations tailored to specific student queries.


    My Contributions:

    • 🔍 Integrating Apache Lucene to enable efficient information retrieval.
    • ✨ Leveraged Microsoft Azure AI Text Analytics for effective text summarisation.
    • 🤖 Configured OpenAI's GPT-3.5 Turbo for empathetic, context-aware AI responses.
    • 💻 Led User Interface (UI) development with Java Swing for an intuitive UX.
    • ℹ️ Developed FAQ page to enhance usability and support access.

    Core Technologies:

    • AI & NLP Tools: Apache Lucene, Microsoft Azure AI Text Analytics, OpenAI GPT-3.5 Turbo
    • Backend Software Development: Java
    • Frontend UI Development: Java Swing
    • Cloud & API Integration: Microsoft Azure AI

    Achievements & Impact:

    • Delivered an AI chatbot for quick, summarised access to university support, helping to reduce student stress
    • Provided a personalised user experience with tailored recommendations for individual student queries
